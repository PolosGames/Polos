# Polos

WIP project.

Roadmap can be found in [this](https://trello.com/b/frG6m0AQ) link.

Currently, the engine only has pre-built Windows binaries for third party
libraries. This will be changed in the feature with building the libraries
per-platform

### Current Features:

- Logging system
- Memory management
- Event Bus (raising events and subscribing / unsubscribing to them)
- A crude window system
- Platform detection

### Soon:

- Rendering API
- Different graphics API supports (Vulkan, DirectX)
