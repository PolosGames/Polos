# Polos

WIP project.

**Roadmap** can be found in [this](https://trello.com/b/frG6m0AQ) link.

**Coding Guidelines** can be found in [Polos Engine C++ Coding Guidelines](https://bit.ly/3YpjIqH).

### To Build

For Windows:

- Configure the build_polos.bat file with your needs, and just run it.

For Linux:

- Configure the build_polos.sh file with your needs, and just run it from the
command line with:

```bash
./build_polos.sh
```

### Current Features:

- Logging system
- Memory management
- Event Bus (raising events and subscribing / unsubscribing to them)
- A crude window system
- Platform detection
- GUI

### Soon:

- Rendering API
- Different graphics API supports (Vulkan, DirectX)
